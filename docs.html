The Problem

Why GhostTrack Exists

Current tracking systems fail in environments with low bandwidth and high latency, leaving commuters without accurate location or ETA data. College campuses are especially affected — poor indoor signal, crowded networks, and unreliable GPS make real-time tracking nearly impossible.

📡

Poor Signal Areas

GPS packets are frequently dropped in dense campus environments, making traditional trackers fail completely with no fallback.

⏱️

Inaccurate ETAs

Without continuous GPS data, ETA predictions become guesswork. Commuters miss buses due to wrong time estimates.

🔌

No Offline Resilience

Systems that depend on constant connectivity crash during network drops instead of gracefully degrading and recovering.

Features

What GhostTrack Does

Every feature is built to handle the worst network conditions while keeping the user experience smooth, informative and real-time.

👻

Ghost Mode

When GPS packets drop, the system switches to predictive mode — using last known velocity to estimate position in real time.

📦

Store & Forward Buffer

Lost GPS points are stored in a buffer. When connectivity returns, they are replayed and merged into the real path automatically.

🧠

ML-Based ETA

Per-bus Linear Regression models trained on live speed and distance data predict arrival times accurately even with sparse updates.

🔍

Anomaly Detection

Isolation Forest ML model detects abnormal bus behavior — sudden speed changes or route deviations — and alerts in real time.

🚦

Delay Detection

Automatic traffic delay detection with toast notifications and banner alerts when a bus slows below threshold speed.

🗺️

Custom Route Setting

Set any start and end point in Bengaluru — the system fetches real road routes using OSRM and updates the bus path live.

📈

Adaptive Updates

Data payload frequency automatically adjusts based on network strength — 1s normal, 2s slow, 3s bad network.

🚏

Stop ETA Predictions

Real-time predictions for each upcoming bus stop — showing which stops are passed and how many minutes to each next stop.

🌙

Dark & Light Map

Toggle between dark and light map themes instantly. All UI panels adapt automatically to maintain readability.
